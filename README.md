# stroke_network_ctmc_simulations

All the files except for "stroke_customization.py" are the codes used to generate results and figures in the paper. Should one need to replicate, run "stroke_main.py" file. "stroke_customization.py" is for customization, i.e., user-specific inputs. Currently, it represents a network of 1 CSC and 3 PSCs and does not scale (in a sense that to add/remove a PSC, one needs to edit the "stroke_functions.py" file.
